CONFIGURAR GIT : 

$ git config --global user.name "John Doe"
$ git config --global user.email johndoe@example.com

Comandos: 
 git config --list: para listar todas las propiedades que Git ha configurado:
 
$ git config user.name  ->  te da el nombre que colocaste
$ git config user.email -> te da el email que colocaste



CREAR REPOSITORIO LOCAL:
1)Inicializar un repositorio localmente:
$ git init nuevo_repo o git init(desde la carpeta)
2)git status vamos a poder ver el estatus de los archivos que tenemos localmente:
$ git status
3)Agregar el o los archivos
$ git add index.html o $ git add . 
4) debemos pasarlos a nuestro repositorio local y para eso debemos usar el git commit:
$ git commit -m "nuevo archivo"


VER HISTORIA DEL REPOSITORIO LOCAL
$ git log (Se vera el autor y la fecha en la que se envió ese commit.)

VER CAMBIOS Y DIFERENCIAS ENTRE COMMITS
1) git log --oneline (obtener los números de commit en version corta a comparar)
2) git diff [version 1] [version 2]


##### RAMAS ##### :  Las ramas son caminos que puede tomar cualquier desarrollador dentro del proyecto, trabajar en el mismo proyecto pero en diferentes funcionalidades

1) Crear una nueva rama o un nuevo branch:
 $ git branch nombrederama
2) Listar las ramas que tenemos:
 $ git branch -l  o  git branch
3) Para borrar una rama:
 $ git branch -D nombrederama
4) Cambiar el nombre a una rama:
 $ git branch -m nombreDeRama nombreDeNuevaRama
5) Movernos entre nuestras diferentes ramas y posicionarnos:
 $ git checkout nombrederama

6°°)Movernos y posicionarnos sobre algún commit específico y desde allí sacar una nueva rama:
a) Primero buscamos y listamos los commits:
$ git log --oneline
b)Con el comando checkout nos posicionamos sobre el commit que seleccionamos:
$ git checkout 95bd15b
c)Podríamos crear una rama e irnos a la misma:
$ git checkout -b nuevonombrederama

### RAMA MERGE #### : FUSIONAR LAS RAMAS
1)Pararnos sobre la rama master y desde allí usar :
 $ git merge ramaAfusionarConMaster
2) Ver diferentes ramas y como se fueron fusionando con nuestra rama master. 
 $ git log --oneline --graph
